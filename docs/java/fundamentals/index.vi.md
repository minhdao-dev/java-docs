## Project cuối phần

**Student Grade Manager** — Console app quản lý điểm sinh viên: thêm/xóa/sửa/tìm, tính GPA, lưu dữ liệu ra file `.txt`. Áp dụng đầy đủ OOP, Comparable, File I/O.

## Điều kiện hoàn thành

- Giải thích được sự khác nhau giữa Stack và Heap
- Viết class hierarchy đúng nguyên tắc OOP
- Override `toString()`, `equals()`, `hashCode()` đúng cách
