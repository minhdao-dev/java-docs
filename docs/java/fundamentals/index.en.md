## Checkpoint

You're ready to move on when you can:

- [ ] Explain the difference between Stack and Heap
- [ ] Use Enum instead of magic strings/ints for fixed constants
- [ ] Read and write files using `java.nio.Files` and `Path`
- [ ] Write a proper class hierarchy following OOP principles
- [ ] Override `toString()`, `equals()`, `hashCode()` correctly
- [ ] Distinguish Abstract Class vs Interface and know when to use each
- [ ] Use Records for immutable data objects instead of writing boilerplate
