## Đầu ra sau phần này

Hoàn thành xong phần này, bạn có thể:

- Gõ `java -version` và thấy đúng JDK 21
- Viết và chạy chương trình Java từ cả IDE lẫn terminal
- Commit và push code lên GitHub
- Hiểu cấu trúc thư mục của một Java Maven project chuẩn

## Thời gian ước tính

Khoảng **3–5 giờ** nếu làm từ đầu đến cuối lần đầu tiên.

## Lưu ý trước khi bắt đầu

!!! warning "Đừng dành cả ngày để setup"
    Cài đặt xong trong một buổi rồi bắt đầu code ngay buổi tiếp theo. Tối ưu theme, font, plugin... là việc có thể làm sau khi đã quen với Java.

!!! tip "Gặp lỗi là bình thường"
    Setup môi trường là bước mà ai cũng gặp lỗi ít nhất một lần. Nếu bị kẹt, đọc kỹ thông báo lỗi, tìm kiếm trên Google với từ khóa chính xác, và xem phần Troubleshooting trong từng bài.
